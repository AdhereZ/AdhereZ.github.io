记录一下偷懒行为

提示

每次发布一篇博客就要将博客代码推到仓库一次，一次就要写三句命令行，我太懒了，就像着能不能写一句命令行就发布成功了，于是乎！

# 编写gitpush.sh文件

git add .
git commit -m 'update'
git push origin main
1
2
3

# 在package.json文件的scripts中添加指令

"push": "bash gitpush.sh",
1

# 最后

每次推送都执行这条命令就行了！

npm run push
